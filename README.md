# Pi Homebrew Tap

Homebrew tap for the Pi CLI.

## Install

```sh
brew tap pi-sloane/tap
brew install sloane
```

## Get started

```sh
sloane login
sloane account
sloane install pi
```

## Formula

The tap provides the `sloane` formula for macOS and Linux.
